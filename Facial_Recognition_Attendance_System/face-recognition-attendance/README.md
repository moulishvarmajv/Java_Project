# Facial Recognition Attendance System

A Java-based **Facial Recognition Attendance System** developed using **Object-Oriented Programming (OOP)** principles and **Maven**. The project aims to automate attendance management through facial recognition while following a clean, modular, and scalable software architecture.

> **Status:** 🚧 Under Development

---

## Features

- Student Management
- Course Management
- Class Session Management
- Attendance Management
- Face Recognition (Upcoming)
- Liveness Detection (Blink & Head Movement) (Upcoming)
- Attendance Reports
- Modular Layered Architecture

---

## Technologies Used

- Java
- Maven
- Object-Oriented Programming (OOP)
- OpenCV *(Upcoming)*
- Haar Cascade Classifier *(Upcoming)*
