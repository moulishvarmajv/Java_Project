# Problems Faced

### Haar Cascade File Missing

Issue:

```
haarcascade_frontalface_default.xml
```

was not present.

Solution:

Downloaded the official Haar Cascade file and placed it inside:

```
src/main/resources/haarcascades/
```

---

### Detection Fluctuation

Issue:

- Multiple faces detected.
- Rectangle occasionally shifted to the eye.
- Face count fluctuated.

Solution:

- Converted frames to grayscale.
- Applied histogram equalization.
- Tuned detectMultiScale() parameters.
- Increased minimum face size.
- Increased minNeighbors.

Detection became significantly more stable.
